## Tile Specifications

- **Tile size**: 16×16 pixels
- **Recommended render scale**: 4× (64×64 pixels on screen)
- **Format**: PNG with RGB colour depth
- **Total tiles**: 128 individual PNG files (`tile_0000.png` – `tile_0127.png`)

## Usage

Import the tile mapping in your component:

```typescript
import { TILE_ASSETS, getTileAsset, TILE_SIZE } from "@/data/tileMapping";

// Render a tile at 4× scale (64px)
const tileUrl = getTileAsset("stone_floor");
```

### Tile size constants

```typescript
TILE_SIZE        // 16  – native pixel size
TILE_RENDER_SIZE // 64  – recommended screen size (4× scale)
TILE_SCALE       //  4  – scale multiplier
```
